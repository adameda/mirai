# Mirai Platform

Projet React minimal prêt à lancer autour du composant existant mirai-platform.jsx.

## Lancer

1. Installer les dépendances : npm install
2. Démarrer le serveur : npm run dev
3. Ouvrir l'adresse affichée par Vite, en général http://localhost:5173

## Modifier

Le plus simple est d'éditer directement mirai-platform.jsx.
Le point d'entrée navigateur est src/main.jsx, qui importe ce composant.

## Scripts

- npm run dev : mode développement
- npm run build : build de production
- npm run preview : aperçu du build
